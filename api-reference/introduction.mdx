## Base URL

All API endpoints are available at:

```
https://your-domain.com/api
```

## Authentication

Currently, the GoTransit API does not require authentication. All endpoints are publicly accessible.

<Note>
Rate limiting may be implemented in the future. We recommend implementing reasonable request throttling in your applications.
</Note>

## Response Format

All successful responses return JSON with the following structure:

- **Success responses**: HTTP 200 with JSON data
- **Error responses**: HTTP 4xx/5xx with error message in JSON format

### Error Response Format

```json
{
  "error": "Error message describing what went wrong"
}
```

## Common HTTP Status Codes

| Status Code | Description |
|-------------|-------------|
| 200 | Success - Request completed successfully |
| 400 | Bad Request - Missing or invalid parameters |
| 404 | Not Found - Station or route not found |
| 500 | Internal Server Error - Server-side error |

## Data Attribution

<Warning>
All data is used with permission of Metrolinx. Please ensure you include the attribution notice in your applications.
</Warning>
