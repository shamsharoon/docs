<Info>
**Prerequisites**:
- Basic understanding of HTTP requests
- Access to a terminal or API client (like Postman or curl)
</Info>

Follow these steps to make your first API call and start building with the GoTransit API.

<Steps>
<Step title="Make your first API call">

Get all available GO Transit routes:

```bash
curl "https://your-domain.com/api/routes"
```

<Check>
You should receive a JSON response with a list of all available routes and their details.
</Check>

</Step>

<Step title="Find stations on a route">

Get all stations for the Lakeshore East route:

```bash
curl "https://your-domain.com/api/route?route=Lakeshore East"
```

<Check>
The response will include all stations on the Lakeshore East route with their stop IDs and sequences.
</Check>

</Step>

<Step title="Get upcoming trips">

Find the next 3 trips from Union Station to Ajax:

```bash
curl "https://your-domain.com/api/next?from=Union Station GO&to=Ajax GO&n=3"
```

<Check>
You'll receive upcoming trip information including departure times, arrival times, and route details.
</Check>

</Step>

<Step title="Use automatic direction detection">

If you have GPS coordinates, you can automatically determine the best direction:

```bash
curl "https://your-domain.com/api/next?auto=1&lat=43.6532&lon=-79.3832&n=3"
```

<Check>
The API will automatically choose between Union Station and Ajax based on your location.
</Check>

</Step>
</Steps>

## Common Use Cases

### Basic Trip Planning

```bash
# Get next 5 trips from Union to Oshawa
curl "https://your-domain.com/api/next?from=Union Station GO&to=Oshawa GO&n=5"
```

### Time-specific Search

```bash
# Find trips departing after 6:00 PM today
curl "https://your-domain.com/api/next?from=Union Station GO&to=Ajax GO&when=2024-01-15T18:00:00-05:00"
```

### Route Exploration

```bash
# See all stations on the Kitchener line
curl "https://your-domain.com/api/route?route=Kitchener"
```

## Response Format

All API responses follow a consistent JSON structure:

```json
{
  "origin": "Union Station GO",
  "destination": "Ajax GO",
  "generated_at": "2024-01-15T14:30:00-05:00",
  "trips": [
    {
      "departure_local": "14:45",
      "arrival_local": "15:23",
      "headsign": "Ajax GO",
      "route": "Lakeshore East",
      "trip_id": "12345",
      "service_id": "weekday",
      "notes": ["scheduled"]
    }
  ],
  "attribution": "Data used with permission of Metrolinx."
}
```
