## Get Started

The GoTransit API provides real-time transit information for GO Transit services across the Greater Toronto Area. Access schedule data, route information, and trip planning capabilities through our RESTful API.

## Key Features

- **Real-time Trip Information**: Get upcoming trips between any two GO Transit stations
- **Route Discovery**: Explore all available routes and their station stops
- **Automatic Direction Detection**: Use GPS coordinates to automatically determine travel direction
- **Flexible Time Queries**: Search for trips at specific times or use current time
- **Multiple Timezone Support**: Work with different timezones as needed

## Data Source

All transit data is sourced from Metrolinx's official GTFS (General Transit Feed Specification) feed and is updated regularly to ensure accuracy.
